Разработать консольное приложение проигрывающее музыку (wav)

1. Хранить список названий треков, исполнителей и путей к ним
2. Добавления трека (название, исполнитель, путь к файлу)
3. воспроизведение трека по номеру (в отдельном потоке)
4. Поиск трека по названию/исполнителю
5. остановка воспроизведения (ввод команды stop)
6. выход по команде exit
